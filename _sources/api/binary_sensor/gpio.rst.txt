GPIO Binary Sensor
==================

Example Usage
-------------

.. code-block:: cpp

    // Basic
    App.make_gpio_binary_sensor("Window Open", 36);
    // Custom pinMode
    App.make_gpio_binary_sensor("Window Open", GPIOInputPin(36, INPUT_PULLUP));

.. cpp:namespace:: nullptr

See :cpp:func:`Application::make_gpio_binary_sensor`.
