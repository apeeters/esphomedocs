PCA9685 PWM¶

FloatOutput support for an PCA9695 16-Channel PWM Driver.

Example Usage¶

// Create the PCA9685 Output hub with frequency 500Hz.
auto *pca9685 = App.make_pca9685_component(500.0f);

See Application::make_pca9685_component().
