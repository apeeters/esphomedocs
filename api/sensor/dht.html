void sensor::DHTComponentset_dht_model(DHTModel model)¶

Manually select the DHT model.

Valid values are:

  • DHT_MODEL_AUTO_DETECT (default)
  • DHT_MODEL_DHT11
  • DHT_MODEL_DHT22
  • DHT_MODEL_AM2302
  • DHT_MODEL_RHT03

Parameters
  • model: The DHT model.
