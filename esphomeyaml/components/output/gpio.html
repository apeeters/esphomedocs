GPIO Output¶

The GPIO output component is quite simple: It exposes a single GPIO pin as an output component. Note that output components are not switches and will not show up in Home Assistant. See GPIO Switch.

# Example configuration entry
output:
  - platform: gpio
    pin: D1
    id: gpio_d1

Configuration variables:¶

  • pin (Required, Pin Schema): The pin to use PWM on.
  • id (Required, ID): The id to use for this output component.
  • All other options from Output.

Warning

This is an output component and will not visible from the frontend. Output components are intermediary components that can be attached to for example lights. To have a GPIO pin in the Home Assistant frontend, please see the GPIO Switch.
